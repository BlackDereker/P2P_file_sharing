3)PROTOCOL
COMMANDS
//////////////////////////
'-' :  command
'*' :  answer to command
//////////////////////////

CLIENT
- HEY 'id'//Gives the id if exists, if not gives '-'
* WELCOME 'id' //answer to client with :1)existing id or the new unique generated id.

- CONN client_ip client_port //Gives the ip and the port of the client
* OK //confirm for correct command

- LIST filea.txt fileb.txt ... //Send to server the names of the files u got
* OK //confirm for correct command

- NAME "name" //gives a fake name for usage
* OK "name"//confirm for correct command

SERVER
- ID 'id' //Gives the unique id to the client
* IAM 'id' //answer from client to server
